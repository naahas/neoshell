# **NEOSHELL EST UN MINI SHELL INTERACTIF RÉALISÉ EN C**

## FONCTIONNEMENT 
#### 1 - Clonez le projet 
#### 2 - Rendez vous dans le répertoire **neoshell*
#### 3 - Compilez le projet et executez le shell en une fois avec **make**


## PREREQUIS
#### **Compilateur GCC pour C**
#### **Outil Make**


## COMMANDES DISPONIBLES
##### *pwd*  (pwd)
##### *show* (ls)
##### *cdir* (mkdir)
##### *rdir* (rmdir)
